## rt file format
### common ground

| Type |  mean  |    |    |    |
|:----:|:----:|:----:|:----:|:----:|
| ``A`` | ambient light | - | ``ratio`` | ``color`` |
| ``L`` | light | ``position`` | ``ratio`` | ``color`` |
| ``C`` | camera | ``position`` | ``normal vector`` | ``fov`` |
| ``#`` | comment | - | - | - |
